# Curiosity-Daily

### 一、简介
该项目为2016级Android生产实习（Group11）项目，开发奇趣日常与设计分享平台。包括三个主要模块，分别为首页模块、设计模块和个人模块。首页模块显示推荐文章与用户关注作者的文章，可查看文章或进行点赞评论，在首页还可进行搜索和发布。设计模块主要为系统推荐设计师创作的壁纸和头像。个人模块中用户可查看个人信息和发布过、点赞过的文章，还可查看使用说明和提交信息反馈。
